\section{Abstract}
\label{abstract}
This thesis investigates how embedded computer vision models designed for soccer analysis can be compressed and deployed efficiently on edge hardware. Specifically, YOLO (You Only Look Once) object detection and pitch keypoint localisation models were analysed, with techniques such as hardware acceleration, input resolution reduction, quantisation, pruning, and knowledge distillation applied to test real-time edge deployment on an NVIDIA Jetson Orin Nano. In total, \textbf{904} model artefacts were generated to explore these techniques and their combinations. Alongside this, a full-stack soccer analysis system was proposed, utilising these models to generate high-level statistics such as formation maps, possession estimation, and pass detection in real time on the edge device.

Results show that hardware acceleration and FP16 quantisation benefited both tasks, reducing object detection latency by \textbf{31.0\%} and \textbf{42.8\%} respectively, and pitch keypoint latency by \textbf{35.5\%} and \textbf{43.1\%} respectively, with negligible accuracy costs in both cases. Object detection retained moderate accuracy under INT8 quantisation, pruning, and input resolution reduction, while pitch keypoint localisation performed substantially worse, indicating greater sensitivity of keypoint models to these more aggressive techniques. Knowledge distillation alone provided little latency benefit to either model task, but worked well in combination with other techniques.

From a hardware perspective, the best compressed models operated well within recommended constraints, with GPU utilisation averaging 40--55\%, power draw 5--15 W, and device temperature 48--56°C (against respective limits of 80\%, 20 W, and 68°C respectively). The best full-stack system achieved a pass mAP$_{1\text{--}5}$ of \textbf{69.7\%}, approaching the SoccerNet baseline of \textbf{71.2\%}, at \textbf{39.6\%} average GPU utilisation and a throughput of approximately \textbf{13.0} FPS without parallelisation or inference interval optimisation. This demonstrates that edge-focused compression can substantially improve the feasibility of embedded soccer analytics, while retaining significant analytical performance in real time.
