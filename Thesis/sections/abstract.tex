\section{Abstract}
\label{abstract}
This work investigates how embedded computer vision models for soccer analysis can be compressed and deployed efficiently on edge hardware. Sppecifically, YOLO object detection and pitch keypoint models were analysed, with various techniques such as hardware acceleration, quantisation, pruning, and knowledge distillation applied with a focus on real-time edge deployment on an NVIDIA Jetson Orin Nano. In total, \textbf{904} model artefacts were generated, exploring these techniques and their combinations. Alongside this, a full-stack soccer analysis system was proposed to utilise these models, generating high-level statistics such as formation maps, possession estimation, and pass detection in real time in the edge device.

Results show that hardware acceleration and FP16 quantisation benefited both tasks, reducing object detection latency by \textbf{31.0\%} and \textbf{42.8\%}, and pitch keypoint latency by \textbf{35.5\%} and \textbf{43.1\%}, with negligible accuracy costs in both cases. Object detection tolerated INT8 quantisation, pruning, and reduced input resolution with a moderate loss in accuracy, while pitch keypoint localisation performed far worse under the same techniques, indicating that keypoint models are more sensitive to aggressive compression. INT8 quantisation reduced pose model mAP$_{50\text{--}95}$ by a significant\textbf{-43.8\%}, and pruning produced similarly poor accuracy relative to object detection models. Knowledge distillation provided no noticeable latency benefit, and was more valuable as a method of speeding up training time.

From a hardware perspective, the compressed models operated well within recommended constraints, with GPU utilisation averaging 40--55\%, power draw 2--5 W, and device temperature 48--56°C (against respective limits of 80\%, 15 W, and 75°C respectively). The best full-stack system achieved a Pass mAP$_{1\text{--}5}$ of \textbf{0.697}, approaching the SoccerNet baseline of \textbf{71.2\%}, at \textbf{39.6\%} average GPU utilisation and a throughput of \textbf{13.0} FPS without parallelisation or inference interval optimisation. This work demonstrates that edge-focused compression can substantially improve the practicality of embedded soccer analytics, while retaining significant analytical performance in real-time.
