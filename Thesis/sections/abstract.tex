\section{Abstract}
This thesis investigates how embedded computer vision models for soccer analysis can be trained, compressed, and deployed efficiently on edge hardware. The work focuses on object detection and pitch keypoint localisation using YOLO11 and YOLO26 model families, with benchmarking carried out on an NVIDIA Jetson Orin Nano Dev Kit. After an initial architecture and dataset selection study, a full experimental pipeline was developed covering baseline training, hardware acceleration, quantisation, pruning, knowledge distillation, and combinations of these techniques. In total, 488 model artefacts were generated and evaluated across the object detection and pose estimation tasks.

The results show that hardware acceleration preserves analytical performance almost exactly while delivering substantial latency reductions, with average improvements of 31.3\% for object detection and 29.9\% for pose estimation. Quantisation produced the largest deployment gains, reducing latency by an average of 56.6\% for object models and 59.6\% for pose models, while also giving the strongest reduction in power usage. However, these benefits came with a larger accuracy cost, particularly for pose estimation. Pruning provided moderate latency improvements for object detection but caused much larger accuracy degradation in the pose task, indicating that the keypoint models were less robust to parameter removal. Knowledge distillation on its own had only a minor effect on latency, but remained useful as a supporting technique for preserving accuracy in smaller deployable models.

These compressed models were then integrated into a full-stack soccer analysis system for player and ball detection, pitch localisation, team assignment, possession estimation, and pass detection. Under the strongest replay setting, the best full-stack configuration achieved a Pass mAP$_{1\text{--}5}$ of 0.697 and 39.61\% GPU utilisation, while reaching an effective throughput of 13.02 FPS on the target device. Overall, the thesis shows that deployment-focused compression can substantially improve the practicality of embedded soccer analytics while retaining useful analytical performance, and that a full computer vision pipeline for match statistic generation can be run effectively on constrained edge hardware.
